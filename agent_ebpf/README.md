# eBPF Backend Agent Scripts (Future)

Hook scripts for the eBPF instrumentation backend.

eBPF-based key extraction is planned for a future phase.

## Structure (Planned)

- `tls/` — TLS key extraction via eBPF uprobes
- `ssh/` — SSH key extraction via eBPF uprobes
- `ipsec/` — IPSec key extraction via eBPF uprobes
